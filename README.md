# PRODIGY_ML_04
# ✋ Hand Gesture Detection System

A real-time **Hand Gesture Recognition System** built using Python and Computer Vision.  
The system detects and classifies hand gestures using a webcam and processes hand landmarks in real-time.

---

## 🚀 Project Overview

This project uses computer vision techniques to track hand movements and identify gestures based on finger positions. It can be extended for automation, smart control systems, and human-computer interaction applications.

---

## ✨ Features

- 🖐️ Real-time hand tracking using webcam
- 🤖 Gesture detection using hand landmarks
- ⚡ Fast and lightweight processing
- 📊 Works in live video stream
- 🎯 Easy to extend for smart applications

---

## 🛠️ Technologies Used

- Python 3
- OpenCV
- MediaPipe
- NumPy

---

## 🧠 How It Works

1. Webcam captures live video feed  
2. MediaPipe detects hand landmarks  
3. Finger positions are analyzed frame-by-frame  
4. Gesture is identified using logic-based rules  
5. Result is processed in real-time output
